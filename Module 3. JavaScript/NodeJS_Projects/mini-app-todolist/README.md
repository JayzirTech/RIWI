# Mini App To-Do List

This is a simple To-Do List application built with vite, tailwindcss and sweetalert2. It allows users to manage their tasks efficiently by adding, marking as completed, and deleting tasks.

## Installation

1. Clone this repository.
2. Run `npm install` to install the dependencies.

## Usage

1. Run `npm run dev` to start the development server.
2. Copy the URL provided in the terminal and open it in your browser.
3. Start adding tasks to your list.

## Features

- Add tasks.
- Delete tasks.
